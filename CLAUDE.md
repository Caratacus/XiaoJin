## 核心指令：语言规范
- **首要语言**：本项目使用**简体中文**作为主要语言。
- **涵盖范围**：包括但不限于：
  - 与用户的所有对话和交流
  - 故事内容创作
  - 文档编写（包括 CLAUDE.md 本身）

## 项目概述

这是一个原创儿童故事系列的创意写作项目，题为"小玉米粒王国英雄小金"。

**主要内容：**
- 205 集已完成的故事，采用 Markdown 格式
- 完整的世界观设定和人物志
- 规范化的创作流程和文档体系

## 架构与约定

### 内容创作流程
1. **阅读设定**：创作前先阅读 `docs/STYLE_GUIDE.md` 和相关 lore 文档
2. **Git notes**：为提交附加详细摘要以供审计

### 文件命名约定
- 故事：`第XXXX集_标题.md`（4位数编号，按顺序排列）
- 世界观设定：`/docs/lore/` 中的描述性名称

### 故事结构（205 集，10 季）
1. **第一季（第 1-11 集）**：生存与重生
2. **第二季（第 12-38 集）**：成长与冒险
3. **第三季（第 39-59 集）**：美食对决与科学启蒙
4. **第四季（第 60-79 集）**：大航海与星际前奏
5. **第五季（第 80-99 集）**：月之谜篇——月球探险与彩虹族复兴
6. **第六季（第 100-122 集）**：星光之旅篇——银河探索与生命/静止共存
7. **第七季（第 123-145 集）**：起源之种篇——追寻宇宙起源真相，成为宇宙守护者
8. **第八季（第 146-165 集）**：万界共鸣篇——学习让不同文明、不同碎片与不同生命形态在边界中彼此倾听，并建立共鸣航线、共鸣灯塔与宇宙驿站
9. **第九季（第 166-185 集）**：地球回响篇——探索可持续的开放模式，建立有边界的宇宙驿站，倾听地球记忆的回响
10. **第十季（第 186-205 集）**：未来之种篇——学习传承与放手，培养新一代守护者，见证春天的延续

### 核心角色简览
详细档案请参阅 [docs/lore/人物志.md](docs/lore/人物志.md)

- **小金**：睿智的英雄、发明家，重生后拥有两世记忆，星芒号船长
- **小玉**：知识守护者、年迈的顾问，现为永恒的记忆体
- **小彩**：彩虹战士，第五季继承彩虹族记忆之种，获得星光之翼；第六季成为星光守护者，实现生命与静止的共存；第七季与起源融合，成为宇宙守护者；第八季从“融合者”成长为真正的“协调者”，学会尊重“不融合”的选择
- **起源**：宇宙诞生时的原初意识，生命与寂静的本源
- **黑焰**：前反派首领，现为王国大厨和星芒号轮机长
- **紫晶国王**：合并王国的统一统治者
- **小翠**：植物学家，星芒号生命系统官
- **银星**：月尘族，地球-月球星际大使

### 第八季摘要
- **季度主题**：和解不是把所有人变成一样，而是让不同的生命愿意彼此倾听。
- **核心变化**：故事重点从”寻找起源碎片”升级为”处理碎片苏醒后，不同文明如何重新学会共处”。
- **关键成果**：
  - 小金提出并实践”观察、沟通、选择”三步机制，开始成为共鸣航线与规则的设计者。
  - 小彩学会区分”想帮助”与”替别人决定”，接受”第二种完整”与”不带走的守护”。
  - 沉默港口白塔从事故遗址转入共鸣灯塔模式，第一段共鸣航线与王国路标建立。
  - 小玉米粒王国正式被多方认可为开放型微观宇宙驿站。
  - 边界议会从敌意与封锁，转向试行”共鸣协约”与多方共签机制。

### 第九季摘要
- **季度主题**：开放需要边界，可持续的开放才是真正的开放。
- **核心设定**：
  - **回响层**：地球记忆的沉积层，通过植物网络显现过去的记忆和创伤
  - **四法则**：分散（不集中承载）、节奏（分时段开放）、边界（分级接入）、传承（记录规则）
  - **共同约定**：六章节十九条款，王国与二十三个文明共同制定的开放规则
  - **三层边界系统**：停靠圈（访客区）、合作圈（交流区）、核心区（居民区）
- **关键事件**：
  - 访客数量激增导致植物网络负载从42%升至87%
  - 回响层因外来数据混入而失控，形成回响风暴
  - 人类调查队首次出现，王国经历隐蔽危机
  - 小金紧急关闭大部分功能，只保留核心运转
  - 二十三个文明代表共同签署《共同约定》
- **角色成长**：
  - 小金从”什么都想接住”到”知道什么必须守住”
  - 学会承认限度不是失败，而是智慧
  - 从单方面制定规则转向共同决策
- **季终状态**：王国进入可持续的开放模式，网络负载稳定在42%，回响层平静，人类活动被持续监测

### 第十季摘要
- **季度主题**：真正的守护不是永远把一切扛在自己肩上，而是在离开之前，让未来也能自己发芽。
- **四阶段结构**：
  - **第一阶段（186-190集）”变化已经开始”**：面对人类灌溉工程，小金提出迁居计划，学会放手
  - **第二阶段（191-195集）”家园要学会迁动”**：协助迁徙，面对失去，学会带着记忆继续前进
  - **第三阶段（196-200集）”未来不能只靠英雄”**：种子学院正式开学，新一代独立成长
  - **第四阶段（201-205集）”未来之种”**：正式交接，小彩创造传承之物，春天继续
- **核心设定**：
  - **种子学院**：培养新一代守护者、记录者、发明者的学院
  - **未来议会**：由王国代表、月尘族、外来文明、植物网络节点组成的决策机构
  - **迁居计划**：23处栖息地、450个族群、2万居民的迁徙工程
  - **未来之种**：凝聚守护精神、可传承、会成长的种子
- **新一代角色**：
  - **小墨**：蚂蚁工程师，负责内部协调，学会独立决策
  - **小星**：瓢虫观察员，负责对外交流，擅长沟通
  - **灰羽**：沉默者后裔，负责应急响应，冷静判断
- **关键领悟**：
  - 守护不是阻止改变，而是帮助大家找到前进的方向
  - 失败也是学习的一部分，有些问题只能被经历
  - 守护不是让一切变得完美，而是让一切变得可以承受
  - 传承是培养能替代自己的人
- **季终状态**：小金和小彩退居顾问角色，新一代正式接手，王国进入可持续传承模式，春天继续

### 提交信息格式
遵循 conventional commits 规范：
```
<type>(<scope>): <description>

类型：feat, fix, docs, style, refactor, test, chore
```

示例：
- `feat(story): 新增第 80 集 - 水晶洞穴`
- `docs(lore): 更新人物档案`
- `fix(story): 修正第10集对话格式`
