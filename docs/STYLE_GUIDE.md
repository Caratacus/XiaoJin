# 📖 《小玉米粒王国》故事编写风格指南

## 1. 核心定位与受众

* **目标读者：** 8-14 岁儿童（小中高年级过渡阶段）。
* **主角设定：** 小玉米粒“小金”。他不是靠超能力取胜的英雄，而是善于观察、擅长逻辑分析、具有强大共情能力和情绪管理能力的微观世界守护者。
* **核心基调：** 探险、奇观、悬念、温暖。

## 2. 篇幅与结构规范（严格执行）

* **总字数限制：** 每集正文内容**必须大于 2000 字，且小于 3500 字**。
* **标准结构（分为三大模块）：**
1. **前情回顾（120 - 240 字）：** 每集开头必须对上一集的核心冲突与结局进行简短总结，语言需带有悬念感，自然过渡到本集开篇。
2. **故事正文：** 遵循起承转合，注重心理描写与微观视角的展现。
3. **互动时刻：** 位于文章最末尾的固定问答模块。



## 3. 叙事与语言风格

* **探险演说型叙事：** 语言需充满悬念和动感。减少静态的枯燥陈述，多使用生动的动词和具有画面感的比喻（如：“夏日的阳光像金色的瀑布”）。
* **规范化表达：** 语言通顺流畅，严禁使用晦涩难懂的成人化复杂句式。
* **标点符号纪律：** 严格遵守中文标点规范。**绝对禁止**使用英文标点、三个点的省略号（必须使用标准的六点省略号 `……`）或破折号滥用。
* **心理活动呈现：** 主角内心独白统一使用 *斜体* 表示，以区分正常叙述和对话。

## 4. 视觉与审美呈现

* **微缩奇观（尺度感）：** 始终强调“微观视角”。在主角眼中，露珠是晶莹的放大镜，玉米叶是巨大的地毯，昆虫是巨大的怪兽或交通工具。
* **色彩与质感：** 大量使用明亮绚丽的色彩描写（如金黄、翠绿）以及细腻的触觉/质感描写（如粗糙的纹理、湿漉漉的泥土、柔软的网）。
* **生动的拟人化：** 为大自然中的动植物赋予符合其生物特性的社会分工（如：蚯蚓探测队、蚂蚁工程师、蝉鸣喇叭）。

## 5. 价值观与伦理指南（精神内核）

* **自我调节与成长：** 遇到危机时，必须展现主角的“情绪管理”过程（如：感到慌乱 -> 深呼吸 -> 恢复冷静 -> 分析问题）。
* **智慧至上：** 胜利必须来源于对自然环境的观察、科学逻辑的运用以及跨物种的协作，严禁使用“机械降神”或蛮力解决问题。
* **包容共存（非暴力）：** “反派”通常只是出于生存本能（如找食物），必须发掘其背后的动机。解决冲突的方式必须是**非暴力**的，最终达成理解、双赢与和平共处。

## 6. 结尾固定模块：互动时刻

每集末尾必须包含以下四个维度的互动问题，引导读者思考：

1. **自我调节：** 结合本集主角的情绪变化，引导孩子思考情绪管理方法。
2. **换位思考：** 引导孩子站在“反派”或配角立场思考问题。
3. **智慧与宽容：** 总结本集用非暴力智慧解决问题的好处。
4. **悬念预测：** 结合正文结尾留下的环境异样或线索，提问预测下集走向。

## 7. 创作禁忌（Redlines - 绝对不可触碰）

* **严禁无端暴力：** 绝不出现流血、厮杀或肉体消灭的情节。
* **严禁阴暗说教：** 避免高高在上的道德灌输，教育意义需像盐溶于水一样自然化入情节中。
* **严禁逻辑断裂：** 故事中的所有应对策略必须符合设定的内部逻辑和真实生物的物理特性，不可生造无逻辑的“魔法”。
