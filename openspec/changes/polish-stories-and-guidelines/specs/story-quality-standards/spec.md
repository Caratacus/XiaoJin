## 新增需求 (ADDED Requirements)

### 需求：标点准确性 (Punctuation Accuracy)
故事内容必须遵循标准语法规则，使用正确的中文标点符号。
- 省略号必须使用六个点（……）。
- 对话必须使用中文双引号（“”）。
- 破折号必须使用两个汉字宽度的长横线（——）。
- 杜绝滥用或混用句号、逗号、感叹号等。

#### 场景：省略号规范检查
- **当** 检查文本中的省略号时
- **那么** 它必须显示为六个点（……），而非三个点或英文句点

#### 场景：对话引用规范
- **当** 角色进行对话时
- **那么** 对话内容必须被包裹在中文双引号（“”）内

### 需求：适龄流畅度 (Age-Appropriate Fluency)
故事内容必须使用适合 7-12 岁儿童阅读的语言风格。
- 句子结构应简洁明了，避免过于复杂的长句。
- 词汇选择应生动形象且易于理解。
- 叙述节奏应通顺自然，符合儿童认知习惯。

#### 场景：句子长度控制
- **当** 检查句子长度时
- **那么** 单个句子（无标点断句）通常不应超过 30 个字

#### 场景：生僻词汇处理
- **当** 遇到较难理解的词汇时
- **那么** 应通过上下文描写进行解释或具体化

### 需求：微观逻辑一致性 (Microscopic Logic Consistency)
故事描述必须始终坚持玉米粒视角的微观尺度感。
- 物理互动需反映微小体型（如：雨滴是巨大的水球，叶子是广阔的平原）。
- 感官细节需匹配微观体验。

#### 场景：雨滴互动描写
- **当** 角色与雨滴互动时
- **那么** 雨滴应被描述为沉重的水球或具有表面张力的物体，而非普通的“雨点”

#### 场景：植物叶片描写
- **当** 描述玉米叶等植物部位时
- **那么** 应将其描述为巨大的绿色滑梯或广阔的地面，符合角色的微小体型
