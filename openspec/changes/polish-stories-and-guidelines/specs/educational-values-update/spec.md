## 新增需求 (ADDED Requirements)

### 需求：教育价值观整合 (Educational Values Integration)
故事指南和内容必须明确整合“自我调节”和“自我认知”作为核心教育主题。
- 故事应展示角色克服内心挣扎或管理情绪的过程。
- 角色应表现出对自我成长和他人的理解。

#### 场景：自我调节能力展示
- **当** 角色面对愤怒或沮丧情绪时
- **那么** 角色应表现出一种能够平复心情或转移注意力的具体策略

#### 场景：自我认知能力展示
- **当** 角色经历失败或犯错时
- **那么** 角色应反思其内在原因，并表现出对自己认知的新成长

### 需求：基于价值观的叙事 (Value-Based Storytelling)
故事必须通过行动和后果来传达积极的价值观（如勇气、诚实、同理心），避免直接说教。
- 价值观应通过情节自然体现。
- 叙述应验证角色的美德行为对社区或自我的积极影响。

#### 场景：隐性价值观传达
- **当** 角色获得某种人生感悟时
- **那么** 这种感悟应源于他们选择的后果，而非其他角色的口头教育

#### 场景：美德行为的正向强化
- **当** 角色表现出符合价值观的行为时
- **那么** 故事应通过正面的结果来强化这一行为的价值
