# 讨厌鬼“铁大钳”与沙子里的金子

小彩开始学习王国的规矩。但他总是太急性子，吃饭时狼吞虎咽，说话时大喊大叫。有一次，他甚至想用新学会的“七彩火焰”来烤玉米，结果差点烧了厨房。小金没有生气，只是耐心地教他：“小彩，慢慢来。力量要用在该用的地方。”

**拦路的“金刚岩”**

上回说到，为了追寻云朵城堡的方向，小金一行人继续向西进发，来到了地鼠首领石头管辖的地下通道。

大家发现，前方唯一的地下水道被一块巨大的黑色怪石堵住了！水流过不去，压力越来越大，发出“咕噜噜”的怒吼。如果不想办法凿开这块石头，不仅通道会被彻底淹没，他们也无法继续西行！

石头首领急得团团转：“这块是‘金刚岩’，太硬了！我的牙齿根本咬不动。只有住在废弃矿坑里的守卫‘铁大钳’能对付它。可是……唉，我真不想去找他。”

**史上最没礼貌的家伙**

为了能继续旅程，小金和石头还是硬着头皮去请来了这位专家。

“铁大钳”是一只长得又黑又丑的蝼蛄（一种擅长挖掘的昆虫）。他刚一出现，小金的眉头就皱成了疙瘩。

铁大钳太讨厌了！

他说话声音像刮锅底一样难听：“哎哟，这不是那群娇滴滴的玉米棒子吗？求我来了？”

他一边走一边乱吐唾沫，还把泥土踢到小金身上，连句对不起都不说。

他嘲笑正在努力挖土的小地鼠们：“笨手笨脚的，你们是在挠痒痒吗？闪开闪开！”

小金气得握紧了拳头，悄悄对石头说：“我最讨厌他了！他太傲慢、太无礼了！等水退了，我一秒钟都不想看到他！”

**令人惊讶的“丑陋爪子”**

“别废话了，看好了！”

铁大钳走到了那块巨大的金刚岩面前。他举起了那对长满了尖刺、看起来非常吓人又丑陋的大前爪。

小金本想转过头不看他，但下一秒，他被震住了。

铁大钳并没有像蛮牛一样乱撞，而是用爪尖在石头上轻轻敲了几下，侧耳听了听声音，嘴里念叨着：“纹理在这儿……受力点在这儿……”

突然，铁大钳大喝一声，那对丑陋的爪子像两把精准的手术刀，以不可思议的角度切入了石头的缝隙。

“咔嚓！”

一声脆响，那块连地鼠牙齿都能崩断的巨石，竟然像豆腐一样整整齐齐地裂开了！积水顺着缝隙哗啦啦地流走了，危机瞬间解除。

**小金的“淘金”时刻**

大家都在欢呼，只有小金愣在原地。他看着铁大钳那副得意洋洋、不可一世的讨厌嘴脸，脑子里却在回放刚才那精彩绝伦的一击。

这时，小玉顾问走到了小金身边，仿佛看穿了他的心思。

“觉得他很讨厌，对吗？”小玉问。

“嗯！超级讨厌！”小金点头，“但是……他敲石头的那个找角度的方法，真的太厉害了。比我们乱挖要快一百倍。”

小玉拿出两个筛子，一个是粗网眼的，一个是细网眼的，笑着说：

“小金，每个人都像一堆沙土。

有的人沙土多（缺点多），比如铁大钳，脾气臭、没礼貌。

但他这堆沙土里，藏着一颗大大的金子（优点），那就是他专注的技术和精准的判断力。

如果你因为讨厌沙子，就把金子也扔掉了，那吃亏的是谁呢？”

小金眼睛一亮：“吃亏的是我自己！因为我也想学会那样凿石头！”

**向“敌人”致敬**

小金深吸一口气，忍住对铁大钳傲慢态度的反感，调整了自己的情绪。他告诉自己：*“我要学习的是他的技术，而不是他的坏脾气。”*

大步走上前去。

“铁大钳先生，”小金认真地说，“虽然我不喜欢你叫我们‘娇滴滴的棒子’，但我必须承认，你刚才找裂缝的那一招简直帅呆了！能不能教教我，你是怎么听出石头纹理的？”

铁大钳愣住了。他习惯了别人讨厌他、怕他，从没见过一个小家伙敢这么直白地说“不喜欢我”却又“崇拜我的技术”。

这只傲慢的蝼蛄尴尬地咳了两声，脸红了：“咳……算你有眼光。其实关键不在力气，在于听回声……”

那天下午，小金一直跟在铁大钳身后。他学会了忽略那些刺耳的脏话（就像过滤掉沙子），贪婪地学习着挖掘的技巧（把金子装进口袋）。

就在他们准备收工时，小金最后一铲子下去，突然感觉到土里有个硬邦邦的东西。

“嘿！铁大钳，你看这是什么？”

小金小心翼翼地刨开沙土，发现那不是金子，而是一张卷起来的、像老枯叶一样坚韧的羊皮纸残片。上面画着弯弯曲曲的红色线条，还有一个若隐若现的云朵标志。

“哼，那是半张古老的藏宝图，”铁大钳瞥了一眼，满不在乎地吐掉嘴里的草根，“我在这里挖了几十年，这玩意儿对我没用。你要是想去送死，就拿走吧。”

小金如获至宝地把它收进了口袋。他感觉到，这半张图或许就是通往“云朵城堡”的关键线索。

**尾声：新的挑战**

小金和铁大钳告别，大家顺利通过了沙地金矿，继续踏上追寻云朵城堡的旅程。

然而，前方又出现了新的挑战。红隼从高空飞来报告：“不好了！通往西边的前方森林变成了红色，所有的树都在发光，像是着火了！而且……而且森林深处传来了‘嘻嘻嘻’的怪笑声！”

“红色的森林？”小金心里一紧，“那会不会是传说中的‘隐形背包森林’？”

**互动时刻**：

1. **情绪调节**：小金虽然很讨厌铁大钳，但他没有让情绪影响自己的判断。如果你不得不和一个你不喜欢的人一起工作或学习，你会怎么做？
2. **自我认知**：小玉告诉小金“每个人都像一堆沙土，里面藏着金子”。你觉得自己身上的“金子”是什么？
3. **发现优点**：铁大钳虽然脾气坏，但技术很好。你能不能试着找出你讨厌的人的一个优点？
4. **悬念预测**：“隐形背包森林”听起来很神秘。你觉得“隐形背包”会是什么？它能装下什么东西？
