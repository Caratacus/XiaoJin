# 第166集：回到露珠河

**前情回顾：** 共鸣风暴平息后，小金和伙伴们终于让不同文明学会了彼此倾听。小玉米粒王国也被正式认可为“开放型微观宇宙驿站”。新航线已经亮起，新的来客随时可能出现。可当星芒号重新转向地球时，小金最惦记的，不是远方的光，而是那条很久没有认真看过的露珠河……

星芒号穿过晨雾时，整片玉米田正被阳光一点点染亮。玉米叶像起伏的绿浪，叶尖的露珠闪着银边。小彩贴在舷窗边，激动得直拍玻璃。

“到家了！那边那根老玉米杆，我一眼就认出来了！”

黑焰笑道：“上次你还把高粱认成塔楼。”

小翠却没有笑，她轻轻吸了口气，小声说：“空气不一样了。还是熟悉的叶香和湿土味，可里面多了一股忙乱的味道。”

小金望着下方的露珠河，心里也微微一动。河道还是那条河道，却多了浮标、新栈桥和花粉灯。岸边空地被修得平平整整，像是正在等什么人停靠。

*原来‘宇宙驿站’这几个字，真的会在家门口长出新模样。*

星芒号刚降落，岸边就响起一片欢呼。年轻玉米粒、蚂蚁小工和光明地鼠团全都跑来了，蜘蛛小姐还特地织了一面亮闪闪的迎宾网。紫晶国王站在最前面，王冠擦得发亮，语气却带着难得的郑重：

“欢迎回来。你们把远方的路带回来了，可王国这几天，也多了不少新问题。”

大家边走边看，变化比想象中更明显。旧装水台改成了双层取水架，广场多了图示牌，露珠河外侧还扩出一圈新空地。石头首领说地下通道已经加修了三条，小翠却摸着湿土皱起眉。

“土层太松了。最近水流是不是变急了？”

紫晶国王点头：“人类换了新的喷灌设备。昨天下午，一股急水差点冲垮东岸新泊位。”

话音刚落，红隼从高处俯冲下来，羽毛上还带着风。

“不只是水。田埂边多了金属杆，夜里会亮白光。外面还有会转头的黑机器。人类正在越来越近。”

小金沉默了片刻。他忽然明白，地球回归并不是回到安稳的旧日子，而是要在更大的变化里重新守住家。

广场中央，一张围着巨露摆开的议事桌已经搭好。桌旁立着几块玉米皮牌子，上面画着取水、停靠和止步的图样。银星也在这时赶到，带来了加固泊位的月光晶核。

“月尘族代表团已经在路上。”她说，“除此之外，还有三份访问问询。真正的第一批来客，很快就会到了。”

刚才还热热闹闹的广场，忽然安静了下来。

紫晶国王没有绕弯子，直接把难题摆上桌面。地方不够，规矩还没定，已经有外地旅人把王国储水露珠当成公共水站来取。居民们一边高兴，一边也担心孩子们的活动区和芽床会被打扰。

“所以现在要决定的，不是欢迎不欢迎。”紫晶国王看向小金，“而是怎么欢迎。”

小金低头望着露珠里晃动的广场倒影。新桥、新牌子、新泊位，还有一张张期待又不安的脸，全都映在那滴水里。

“先把家和驿站分开。”他终于开口，“露珠河外侧先做接待区，泊位、饮水、翻译和休息都放在那里。再往里，仍然是居民生活区。欢迎别人，不等于没有边界。边界不是拒绝，而是保护彼此最重要的地方。”

黑焰立刻点头：“像厨房的外备菜台。客人先坐稳，真正的火口不能随便碰。”

小翠也赞成：“生态区必须分层，不然善意也可能变成伤害。”

独眼警卫补充：“建议尽快建立基础翻译节点和访客权限。”

小彩拍桌而起：“那就先把外圈立起来！”

紧绷的气氛刚松一点，独眼警卫忽然发出警报：“东南方，强反光接近！”

红隼瞬间升空，下一秒就在高处大喊：“是新喷灌头！它正对着露珠河上游转过来！”

所有人脸色一变。小金几步跃上桌边，一眼就看见叶海尽头那片冰冷的银光。粗水管已经开始震动，如果第一股水正冲河口，刚修好的泊位和月光晶核都会被卷走。

“别乱跑！”小金立刻下令，“石头首领，加固东岸土坡。蜘蛛小姐，把缓冲网拉到泊位外侧。黑焰，护住晶核。小彩，跟我去上游！”

广场瞬间动了起来。地鼠团钻入地下，蜘蛛丝在风里绷成亮弧，黑焰和银星、小翠一起护住晶箱。小彩展开翅膀，带着小金逆着露珠河冲向上游。

喷灌头立在田埂边，在他们眼里像一座会转动的银塔。水管深处传来低沉闷响。小金扫了一眼四周，忽然看见一截修栈桥剩下的空种壳和一根细藤。

“不拦水，拦不住。”他飞快说道，“把种壳卡进泥槽，用藤绳把喷头偏开一点，只要第一波不正对河口，下面就撑得住！”

小彩立刻去搬种壳。小金扑进湿泥里，把藤绳绕上喷头底座。震动越来越强，泥点溅得他满腿都是。

“放这边！”

两人一推一卡，种壳“咔”地嵌进泥槽。小彩死死拉住藤绳，翅膀都绷直了。轰的一声，第一股水柱猛地喷出，像一堵透明高墙朝下砸来。好在喷头被硬生生拉偏了一角，主水流擦着河口冲进杂草地，碎水四散飞溅。

小金抬头望向下游。东岸没有被正面击中，泊位还在。远处的缓冲网和土坡已经接住了散开的余水。

小彩抹掉脸上的水，大口喘气：“成了！”

小金也笑了一下，可很快又收住了。他看着那道还在喷涌的新水流，心里反而更清楚了。

这只是开始。

以后要来的，不只是客人，还有越来越近的人类设备、越来越复杂的规则，和每一天都要认真照看的边界。

等水势慢慢缓下来，阳光已经完全照亮广场。大家重新聚到议事桌边，鞋底和叶坡都闪着亮晶晶的水痕。紫晶国王看着完好的泊位，长长松了口气。

“看来，驿站的第一条规矩，不该写在牌子上。”他说，“而该写在心里：不是所有变化，都能等它来了再准备。”

小金望着重新流稳的露珠河，轻轻点头。

“星芒号回家了。”他说，“但从今天起，家园也要学会迎接更大的世界。”

露珠河依旧在晨光里流动。只是这一次，它流向的不再只是熟悉的昨天。

**互动时刻：**

1. **自我调节：** 喷灌头突然转向时，大家都很紧张。小金先看清危险从哪里来，再安排每个人做什么。你遇到突然变化时，会怎样先让自己冷静下来？

2. **换位思考：** 王国居民既想欢迎新朋友，又担心自己的生活被打乱。如果你是本地居民，你最担心什么？如果你是远道而来的客人，你希望主人怎样接待你？

3. **智慧与宽容：** 小金提出“把家和驿站分开”，不是把客人挡在外面，而是让大家都更安心。为什么真正长久的欢迎，往往需要清楚的边界？

4. **悬念预测：** 月尘族代表团和其他访客很快就要抵达，而人类的新设备也在继续靠近。接下来，露珠河边的第一批来客，会带来怎样的新惊喜和新麻烦呢？
