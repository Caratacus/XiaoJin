# 急性子的“七彩弟弟”与回声荆棘

**突然冒出来的“小火箭”**

小金背着装满“耐心之水”的水壶，正站在分岔路口思考该走哪条路。

左边是一条“花香小径”，路很平坦，但弯弯曲曲的，看起来要走很久。

右边是一条“黑漆漆隧道”，看起来笔直通向家门，但里面阴森森的。

就在这时，远处传来一阵急促的脚步声：“哥哥！哥哥！我来接你啦！”

一个长得五颜六色的小玉米像一颗小炮弹一样冲了过来。他是“小彩”，那个几个月前从小金带回来的“彩虹种子”里蹦出来的小家伙。

当那颗沉睡在水底千年的种子在小金的实验室里第一次睁开眼时，它看到的第一眼就是小金正温柔地为它擦拭外壳。从那天起，这个精力旺盛的小家伙就成了小金最忠实的跟班，整天“哥哥、哥哥”地叫个不停。虽然他们没有血缘关系，但在小金心里，这个由他亲手唤醒的彩虹少年，就是他最亲的弟弟。

“哥哥，听说你带了宝物回来！快！我们走这条直路回家，我要马上看宝物！”

小彩连气都没喘匀，拉着小金就要往那条“黑漆漆隧道”里钻。

**小金的“慢镜头”**

小金轻轻拉住了弟弟，温和地说：“小彩，别急。你看，那条隧道口长满了带刺的植物，而且没有光。虽然它短，但可能很危险。我们还是走花香小径吧，虽然慢一点，但很安全。”

“哎呀！哥哥你太磨蹭了！”小彩把嘴巴撅得高高的，像个挂油瓶，“我才不怕黑！我是勇敢的彩虹战士！那条花路要走一百年呢，我才不要等！”

说完，任性的小彩根本不听劝，挣脱了小金的手，像个小旋风一样冲进了黑漆漆的隧道。

“哼，我要证明我是最快的！”

**越挣扎越紧的“坏东西”**

小金没办法，只好叫上红隼，赶紧追了进去。

刚跑进隧道没多远，就听见前面传来了小彩愤怒的吼叫声：“放开我！你们这些坏东西！我踢死你们！”

原来，这条隧道里长满了“回声荆棘”。这种植物有个坏脾气：你越是急躁、越是暴力对待它，它就缠得越紧。

小彩刚才跑得太急，被一根藤蔓绊倒了。他一看这藤蔓敢挡路，心里的火气“蹭”地一下就上来了（他最讨厌坏东西挡路了）。

“敢挡本少爷的路？”小彩用力去扯，还用脚去踢，大喊大叫。

结果，荆棘感受到了他的怒气，瞬间活了过来，变出了更多的刺，像包粽子一样把小彩捆得结结实实，连嘴巴都快封住了。

**哥哥的“温柔武器”**

“哥哥……救命……”小彩动弹不得，急得眼泪在眼眶里打转，但他还是不服气，“这破草太坏了！我要拿火烧了它！”

这时，小金赶到了。

他看到这一幕，并没有慌张，也没有责怪弟弟。

他先停下来，观察了一下荆棘的纹理（这是在地下矿坑学的专注），然后从背包里拿出那壶“耐心之水”。

小金没有像小彩那样去攻击荆棘，而是单膝跪地，用非常有礼貌的声音对荆棘说：

“对不起，荆棘先生。我弟弟太着急回家了，不小心撞到了您。他不是故意的，请您原谅他的鲁莽。”

说完，小金倒出一点点清凉的耐心之水，轻轻淋在那些紧绷的藤蔓上。

**小彩的“大眼睛”**

神奇的事情发生了。

喝到了耐心之水，又听到了温和的道歉，那些暴躁的荆棘像被抚摸的小猫一样，慢慢软化了，刺也收了回去。它们缓缓松开，把小彩放了出来，甚至还弯下腰，为小金指明了出口的方向。

小彩惊呆了。他睁着那双五颜六色的大眼睛，看着哥哥。

他原本以为，对付“坏蛋”就要比它更凶、更狠。

可哥哥刚才什么武器都没用，只用了“礼貌”和“耐心”，就打败了这些可怕的家伙。

**榜样的力量**

走出隧道后，小彩低着头，脸红红的。

“哥哥……我是不是又搞砸了？”

小金蹲下来，帮弟弟拍掉身上的灰尘，笑着说：“小彩，你有勇气冲进黑暗，这很棒。但是，真正的强者不是拳头最硬的人，而是能控制住自己脾气的人。有时候，‘慢’就是‘快’，‘温柔’就是‘力量’。”

小彩看着哥哥沉稳的样子，用力地点了点头。他突然觉得，哥哥比那个只会乱冲乱撞的自己帅多了。

“哥哥，我也要学那个‘把坏脾气变成礼貌’的魔法，你能教我吗？”

小金牵起弟弟的手：“当然可以。我们一起走回家吧，这次，我们慢慢走。”

**尾声：奇怪的森林**

七彩弟弟终于学会了“慢慢来”，他的根扎得更深了，长得也更茁壮了。

告别了种子旅人们，小金一行继续前进。然而，前方出现了一片奇异的森林——那里的荆棘会说话，而且专门喜欢说反话！

“你不该往这边走！”一根荆藤大声说道。

小金停下脚步：“往哪边走才对呢？”

“当然不是那边！”另一根荆藤指着相反的方向。

小金困惑了：“那到底该往哪边？”

“反正不是这边也不是那边！”第三根荆藤咯咯地笑了起来。

小玉无奈地摇摇头：“这是‘回声荆棘林’，它们最爱说反话，真不知道要怎么才能通过这里。”

就在这时，一个只会说“不知道”的身影从树林深处走了出来……

**互动时刻**：

1. **情绪调节**：小彩被荆棘缠住时很生气，结果越缠越紧。你在生活中有没有遇到过越生气越糟糕的事情？
2. **自我认知**：小彩一开始觉得“快就是好”，后来明白了“慢就是快”。你觉得什么时候需要快，什么时候需要慢？
3. **解决问题**：小金用礼貌和耐心解决了暴力解决不了的问题。下次遇到冲突，你愿不愿意试试小金的方法？
4. **悬念预测**：那个只会说“不知道”的身影是谁？他真的什么都不知道吗？
