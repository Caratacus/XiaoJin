# 嫉妒的荆棘林与“赞美的魔法”

谁是世界上最漂亮的？

小金和小彩来到了寻找绿色宝石的地方——“幽暗荆棘林”。

这里没有阳光，四周长满了墨绿色的藤蔓和带刺的灌木。空气冷冰冰的，每一片叶子都像是尖尖的刀片。

小彩今天穿着他最喜欢的七彩鳞片战衣，在昏暗的森林里闪闪发光，简直像个移动的迪斯厅灯球。

他一边走，一边嫌弃地看着周围：“哎呀，这里真丑！黑漆漆、绿惨惨的。还是我最漂亮！你们看我的衣服，多亮！多美！”

小彩像只骄傲的小孔雀，故意在灌木丛前转了个圈，展示他的光彩。

被“围攻”的孔雀

就在小彩炫耀的时候，森林突然“活”了过来！

周围的荆棘藤蔓仿佛听懂了他的话，生气地颤抖起来。它们从四面八方伸出来，发出嘶嘶的声音：

“你漂亮？你笑话我们丑？”

“讨厌的家伙！爱现的家伙！我们要遮住你的光！”

“嗖！嗖！”

无数根带刺的藤蔓像鞭子一样抽向小彩。它们不想伤害他，但想用满是泥浆的叶子把他裹住，不想让他那么显摆。

“哎呀！别抓我！我的新衣服！”

小彩吓得哇哇大叫，但他越是挣扎，越是大喊“我就是比你们好看”，藤蔓就缠得越紧，尖刺把他的漂亮衣服都划破了。

小金的“隐身术”

“红隼，快去帮忙解围！”

小金没有像小彩那样发光，他穿着朴素的草叶外衣，几乎和森林融为了一体。

小金冲过去，但他做的第一件事，不是砍断藤蔓，而是从地上抓起一把黑泥巴，啪的一下，抹在了小彩发光的脸上和衣服上！

“哥！你干嘛！脏死了！”小彩惊呆了。

“嘘！别说话！把你的光芒收起来！”小金严厉地低声说。

随着小彩变得灰扑扑、不再刺眼，那些疯狂攻击的藤蔓动作迟疑了一下，似乎找不到攻击目标了。

赞美是最好的除草剂

小金挡在狼狈的小彩面前，对着周围愤怒的荆棘，深深地鞠了一躬。

他没有炫耀自己，而是真诚地看着那些墨绿色的叶子，赞叹道：

“荆棘先生，灌木女士，其实你们非常美。

看这深沉的墨绿色，多么有生命力啊！

看这些尖锐的刺，是多么强大的保护伞啊！

如果没有你们在黑暗中守护森林，这里早就干枯了。你们才是这里的卫士。”

绿宝石的绽放

听到小金的话，那些原本张牙舞爪的藤蔓愣住了。

它们习惯了被嫌弃“丑陋”、“阴暗”，从来没有人夸过它们“有生命力”和“强大”。

藤蔓们害羞了，原本墨绿色的叶子竟然泛起了一层柔和的嫩绿光泽。

“他……他说我们美……”

“他说我们是卫士……”

“沙沙沙——”

荆棘藤蔓慢慢退去了，它们不再攻击，而是主动向两边分开，让出了一条铺满苔藓的小路。

在小路尽头，一朵巨大的绿色食人花缓缓张开嘴巴，吐出了一颗晶莹剔透的绿色魔法宝石——那是森林对谦虚者的馈赠。

真正的“漂亮”

走出森林后，小彩看着自己满身的泥巴和破衣服，又看了看手里珍贵的绿宝石，若有所思。

小金帮弟弟擦掉脸上的泥，温柔地说：

“小彩，你本来就很优秀、很漂亮，这不需要通过‘贬低别人’来证明。

当你太爱炫耀时，别人的嫉妒就会变成刺来扎你；

当你学会收敛光芒，去夸奖别人时，别人的刺就会变成花朵来送给你。”

小彩点点头：“哥，我懂了。刚才那些藤蔓其实挺可怜的，它们只是觉得自己不够好。以后我再也不乱显摆了，我要学会发现别人的漂亮！”

给爸爸妈妈的“故事解码”：

这一章教会孩子“谦虚”与“社交智慧”：

1. 为什么会受排挤？ 很多孩子不明白为什么自己穿了新衣服、拿了新玩具，小伙伴反而不理他了。故事告诉他：“像小彩那样过度炫耀，会引来‘嫉妒藤蔓’。”

2. 如何化解嫉妒？ 小金的做法是教科书级的：降低姿态（抹泥巴/谦虚） + 真诚赞美对方。

3. 核心价值观： 告诉孩子：“真正的自信，不需要通过踩低别人来体现。能看到别人优点的人，才是发着光的人。”

嫉妒的荆棘林在"赞美的魔法"下变得柔软了，荆棘们也学会了为别人的优点感到高兴，而不是嫉妒。

告别了嫉妒的荆棘林，大家来到了一座巨大的图书馆。然而，这座图书馆有个奇怪的规定——必须保持绝对安静，连呼吸声都不能太大！

"嘘——"图书管理员——一只老猫头鹰竖起手指，"这里是'静悄悄图书馆'，大声说话会被请出去的！"

小金小心翼翼地走进图书馆，却发现了一个问题：图书馆里有很多需要小声讨论的事情，可大家都憋着不敢说话，效率特别低。

"这样不行，"小金小声对伙伴们说，"我们需要找到一个既能安静又能交流的方法。"

就在这时，图书馆里突然传来了一声巨大的"哈——欠——！"打破了寂静...

**思考与探索**：

1. 嫉妒的荆棘为什么有攻击性？它们真正需要的是什么？
2. "赞美是一种魔法，能让尖锐变柔软"，这句话对你有什么启发？
3. 小金是如何化解荆棘林的嫉妒的？
4. 故事结尾提到"静悄悄图书馆"和那个巨大的哈欠，你觉得会发生什么？
